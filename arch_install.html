Instalação Básica de Sistemas Operacionais - Arch Linux

Instalação Alvo

Como o Arch Linux é bastante configurável, toda instalação tem um alvo de como ser finalizada. No caso dessa instalação trata-se de um sistema em UEFI com systemd-boot, e iwd+resolved ou NetworkManager+resolved.

ALERTA! Necessário um mínimo de experiência prévia com sistemas Unixlike.

//SELECIONAR TECLADO ABNT2

> loadkeys br-abnt2

//SELECIONAR FONTE TERMINUS BOLD (opcional)

> setfont ter-124b

//CONFIGURAR REDE

para Ethernet

Verificar aquisição de IP e conexão com a rede:

> ip address show

> ping google.com

para Wifi

> iwctl

> device list

> station "device" scan

> station "device" get-networks

> station "device" connect "SSID"

> exit

Verificar aquisição de IP e conexão com a rede:

> ip addr show

> ping google.com

//EDITAR MIRRORS COM O VIM

> vim /etc/pacman.d/mirrorlist

> :%s/Server/#Server/g

Descomente os servidores brasileiros deletando o "#"

> :wq

//ATUALIZAR BASE DE PACOTES

> pacman -Syy

//PARTICIONAMENTO

> fdisk -l (para verificar os discos)

> gdisk /dev/sdx (para selecionar o disco a ser particionado, substitua o x pela letra do seu disco)

> o (para criar uma nova tabela de partição em GPT)

partição EFI

> n

> partition number: default (tecle Enter)

> first sector: default (tecle Enter)

> last sector: +512M

> type: ef00

para partição de swap (opcional)

> n

> partition number: default (tecle Enter)

> first sector: default (tecle Enter)

> last sector: +3G (para 3 gigas de swap, verifique quanto de swap é indicado para seu computador com relação a essa tabela abaixo

> type: 8200

para / (ou para /home)

> n

> partition number: default (tecle Enter)

> first sector: default (tecle Enter)

> last sector: +100G (para uma partição de 100gb, por exemplo) ou tecle Enter (para usar até o fim do disco)

> type: 8300 (ou tecle Enter)

gravar e sair

> w

//VERIFICAR PARTIÇÕES

> lsblk

//FORMATAR PARTIÇÕES

> mkfs.fat -F32 /dev/sda1 (considerando o seu disco como /dev/sda e 1 como o número da partição efi)

> mkswap /dev/sda2 (caso use partição swap e considerando o seu disco como /dev/sda e 2 como o número da partição swap)

> mkfs.ext4 /dev/sda3 (considerando o seu disco como /dev/sda e 3 como o número da partição raiz, a /home pode ser formatada com o mesmo comando mkfs.ext4)

//MONTAR PARTIÇÕES (para systemd-boot)

> mount /dev/sdxy /mnt (altere o x para a letra do disco e o y para o número da partição raiz)

> swapon /dev/sdxy (caso tenha swap em partição no disco altere o x para a letra do disco e o y para a partição swap)

> mkdir /mnt/boot

> mount /dev/sdxy /mnt/boot (altere o x para a letra do disco e o y para o número da partição efi)

//INSTALAR SISTEMA

> pacstrap /mnt base linux (ou linux-lts) linux-firmware iwd (caso utilize wifi) nano (sinta-se a vontade para instalar o vim no lugar do nano caso deseje) networkmanager (caso queira um gerenciamento de rede gráfico e mais integrado) intel-ucode (ou) amd-ucode sudo (sim, o sudo é um programa que deve ser instalado se você deseja utilizar elevação de privilégios)

//GERAR FSTAB

> genfstab -U /mnt >> /mnt/etc/fstab

Verificar fstab

> cat /mnt/etc/fstab

//CHROOT PARA INSTALAÇÃO

> arch-chroot /mnt

Para swapfile (apenas caso for utilizar ao invés do swap por partição)

> fallocate -l 3GB /swapfile

> chmod 600 /swapfile

> mkswap /swapfile

> swapon /swapfile

> nano /etc/fstab

Adicione no final do arquivo:

/swapfile none swap defaults 0 0

//CONFIGURAR TECLADO ABNT2

> echo KEYMAP=br-abnt2 >> /etc/vconsole.conf

//SETAR TIMEZONE

> ln -sf /usr/share/zoneinfo/America/Sao_Paulo /etc/localtime

> hwclock --systohc

//SETAR LOCALE

> nano /etc/locale.gen

Habilite “pt_BR.UTF-8 UTF-8” descomentando a linha deletando o "#"

> locale-gen

> echo "LANG=pt_BR.UTF-8" >> /etc/locale.conf

//CONFIGURAÇÕES DE REDE

Selecione o nome do host

> echo meuhostname >> /etc/hostname

Edite /etc/hosts

> nano /etc/hosts

Adicione:

127.0.0.1 localhost

::1 localhost

127.0.1.1 meuhostname.localdomain meuhostname

//ADICIONAR SENHA DE ADMINISTRADOR

> passwd

//INSTALAR BOOTLOADER E INICIAR REDE

> bootctl --path=/boot install

> nano /boot/loader/loader.conf

Adicione:

default arch.conf

> nano /boot/loader/entries/arch.conf

Adicione as linhas:

title Archlinux

linux /vmlinuz-linux

initrd /intel-ucode.img (ou amd-ucode)

initrd /initramfs-linux.img

options root=/dev/sdxy(raiz) rw

//HABILITAR SERVIÇOS DE REDE

systemctl enable systemd-networkd.service (para systemd-networkd como gerenciador de rede)

Ou

systemctl enable NetworkManager.service (para NetworkManager como gerenciador de rede)

systemctl enable systemd-resolved.service (para solução de DNS pelo systemd-resolved)

systemctl enable iwd.service (para wifi com iwd)

Nota: Você pode utilizar o iwd em conjunto com o systemd-resolved para uma configuração com wifi mais simples (Sway, por exemplo) já que ele possui resolução interna de DHCP ou utilizar o Network Manager com o systemd-resolved para uma configuração mais padrão (Desktop Environment e cabo).

//CRIAR USUÁRIO COMUM E HABILITANDO SUDO

> useradd -mG wheel seunomedeusuário

> passwd seunomedeusuário

> nano /etc/sudoers

Habilite a elevação de privilégio descomentando a seguinte linha

%wheel ALL=(ALL) ALL

//FINALIZANDO INSTALAÇÃO

> exit

> umount /mnt/boot

> swapoff (caso utilize swap)

> umount /mnt

> reboot

//RETOQUES PÓS INSTALAÇÃO

Habilite o TRIM periódico para o SSD:

> systemctl enable fstrim.timer

Habilite cor no pacman:

> nano /etc/pacman.conf

Descomente a linha "Color"

Habilite noatime:

> nano /etc/fstab

Mude de:

UUID=XXXXXXXXXXXXXXXXXXXX / ext4 relatime,errors=remount-ro 0 1

Para:

UUID=XXXXXXXXXXXXXXXXXXXX / ext4 noatime,errors=remount-ro 0 1

//INSTALAÇÃO DO CINNAMON

> pacman -S xorg cinnamon lightdm-gtk-greeter lightdm-gtk-greeter-settings filemanager-actions gnome-terminal firefox

> systemctl enable lightdm

Para habilitar o Folder Color no Nemo:

> pacman -S install base-devel git nemo-python

> git clone https://aur.archlinux.org/yay.git

> cd yay

> makepkg -si

> yay -S mint-x-icons mint-y-icons folder-color-switcher

> nemo -q

Para habilitar extração/compressão no menu de contexto do Nemo:

> sudo pacman -S nemo-fileroller

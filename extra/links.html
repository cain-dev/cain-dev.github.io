Ferramentas para computadores retro

Agradecimentos ao ssshake

TheOldNet.com

A partir de 2019, tive a ideia de poder navegar na web da década de 1990 em alguns dos meus computadores da década de 1990. Para minha surpresa, os arquivos online existentes não eram compatíveis com navegadores antigos, embora o conteúdo fosse daquela época. Num fim de semana, escrevi um proxy e todo o resto cresceu a partir daí. Dois anos depois, forneci quase uma dúzia de serviços gratuitos para entusiastas de computadores antigos em todo o mundo!

Protoweb

Este é um proxy HTTP que tem como objetivo a integridade de sites de alta qualidade. A equipe da protoweb coleta o Internet Archive e outros recursos e remonta os sites manualmente da maneira mais completa possível. Você não encontrará todos os sites no Protoweb, mas os que encontrar serão da melhor versão que existe.

Retrocampus BBS

Este BBS tem de tudo, jogos, chat, informações modernas de sites atuais. O melhor de tudo é que é compatível com mais de uma dúzia de sistemas!

68k.news

Este é um proxy para o Google News que retorna texto e imagens, mas nenhuma das coisas incompatíveis que os navegadores antigos não gostariam.

FrogFind!

O mecanismo de pesquisa para computadores antigos.

Wiby

Wiby é um motor de busca com o objetivo declarado de “construir uma teia de páginas como era nos primórdios da internet”. Possui um grande índice de páginas contribuídas leves, um recurso aleatório "surpreenda-me..." e um filtro somente HTTP. Ele também está hospedado em uma frota de Optiplexes e abriga uma sala de bate-papo sem script.

WRP (Web Rendering Proxy)

Este é um serviço muito inteligente e legal. Ele permite que você navegue na Internet moderna em computadores antigos. Geralmente, isso é impossível porque a maioria dos sites é parcial/totalmente renderizada usando javascript do lado do cliente. Os navegadores antigos não conseguem lidar com isso. O WRP pré-renderiza as páginas da web em uma instância headless chrome e, em seguida, retorna uma captura de tela da página da web como um mapa de imagem para o computador antigo. Maneira muito legal de resolver o problema.

Browservice

Este é um proxy pré-renderizado semelhante ao WRP. Ele fornece uma experiência mais tranquila em detrimento da compatibilidade, pois requer uma versão mínima de javascript para funcionar.

Retro-Proxy

Um proxy HTTP auto-hospedado que remove SSL e javascript.

Netscape Proxy

Este é um proxy pré-renderizado semelhante ao Browservice. Foi criado pelo The Eric Experiment para resolver a falta de suporte do Netscape por parte do Browservice. Este proxy não é tão maduro quanto o Browservice ou o Web Renderer Proxy, pois é relativamente novo.

WebOne

Este é um servidor proxy HTTP 1.x que torna antigos navegadores, reprodutores de mídia e mensageiros utilizáveis ​​novamente no mundo da Web 2.0.

Timeprox

Um proxy HTTP Node.js para a Wayback Machine para navegadores retro.

Kali

Kali era um programa usado na era do Windows 95 para fazer jogos locais/IPX pela Internet. É um cliente/servidor de lobby de jogo e um proxy. É bastante impressionante e ainda está em operação hoje! Este serviço foi importante à medida que os jogos faziam a transição do multijogador local/discado para o multijogador TCP/IP na Internet. Muitos jogos populares da época não podiam ser disputados diretamente pela Internet. Esse proxy permitiu que esses jogos se conectassem pela Internet, o que foi incrível.

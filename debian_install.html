Instalação do Debian

Alerta: Esse tutorial presume que você viu a página de Introdução de Instalação de Sistemas, acessou os links para download das isos e já passou ela para um pendrive através de um dos métodos descritos! Caso não tenha feito isso, clique aqui para ser direcionado para a Introdução de Instalação de Sistemas.

Introdução

Era 2005, tempos que um computador com Celeron D de 2ghz, 40 de HD e 256 de RAM custava 4K! Quase todo mundo com o famoso "pc branco da Positivo" que veio com alguma localização de Linux (mas que você mudou pro Win xispezão piratex que eu sei) acessando Orkut e MSN (saudades do MSN), e é muito provável que você manteve essa usina de 90nm até meados de 2012 quando você montou um segundo pc de mesa já considerando upgrade futuro ou (o caso mais comum) comprou aquele notebook de entrada com APU AMD com o famoso "combo da desgraça": 2 núcleos, 2ghz e 2gb de ram. Hoje em dia? Abre nem o Chrome mais sem travar.

Seja lá qual for a sua história, sempre tem aquele computador, seja de mesa ou notebook, seja que você comprou e ficou guardado ou a sua tia que te deu, que AINDA funciona! E que você poderia usar nem que fosse pra baixar filmes ou escutar música, mas que fazer essa porcaria funcionar num Windows 7 atual é como pedir perdão pra receita federal ou tentar justificar algo pra uma mulher na TPM.

A maior parte dos sistemas está simplesmente abandonando os pcs de arquitetura 32 bits... E agora? Será que existe algum sistema com suporte pra 32 bits e que faça essa carroça funcionar minimamente bem? Tem sim! E ele se chama Debian!

Se você conhece um mínimo do universo Linux o Debian não precisa de apresentações, mas se você não é desse universo, basta entender que o Debian é um distribuição multiarquitetura com muito suporte e estabilidade que é tão influente e referência que chega a ser chamado de "pai de todos" pois tem uma absurda quantidade de distribuições de Linux que são baseadas nele.

O Debian definitivamente não é uma distribuição para iniciantes! Ela se foca em atender servidores e não usuários finais... O que não quer dizer que não vai rodar muito melhor e mais seguro no seu pc do que aquela porcaria de Windows 7 32 bits Home crackeado que você tem a cara de pau de chamar de sistema.

Aqui você não vai ter lojinha e 1001 aplicações para "amortecer" a sua experiência com o Linux, mas pra um pc velho? Nem precisa pois já vem com quase tudo instalado. Vamos começar?

A instalação alvo aqui é por meio da mídia oficial com NetInstall justamente devido a leveza, a instalação será feita com cabo de rede (mas basta configurar o seu WiFi durante a instalação também que tá tudo certo) e graficamente (para ser mais fácil já que basta clicar na opção que deseja e depois em continuar) mas caso queira um processo de instalação ainda mais rápido você pode simplesmente selecionar a opção Install ao invés de Graphical Install e seguir esse mesmo tutorial já que as opções são basicamente as mesmas e basta usar as Setas do teclado para navegar, Espaço para marcar e desmarcar caixas de diálogo, Enter para confirmar e Tab para alterar entre as opções.

Instalação

Em Selecione o seu idioma vamos no Português do Brazil no nosso caso.

Em Selecionar sua localidade vamos de Brasil (mas se você tá lendo isso de Portugal abraços 😄)

Em Configure o teclado vamos de Português Brasileiro para ABNT2

Em Configurar a rede coloque o nome para identificar seu computador na sua rede local, eu recomendo colocar marca do computador + número do modelo (como por exemplo asus-h61m)... Mas nada impede de chamar o seu computador de "Joãozinho" se você quiser...

Nessa tela escreva o nome do domínio mas se você não tem domínio ou nem sabe o que é isso basta deixar em branco

Aqui defina a senha da conta administrativa do computador, recomendo que seja uma senha forte de pelo menos 8 caractéres com maiúsculas, minúsculas, números e caracteres especiais

Aqui é para inserir seu nome real e completo... Mas nem precisa então pode deixar em branco mesmo

Insira o nome da conta de usuário

Insira e confirme a senha para esse usuário padrão

Selecione a sua região de fuso horário

Aqui você seleciona como quer particionar o disco, eu recomendo escolher Assistido - usar o disco inteiro a menos que você saiba muito bem o que esteja fazendo

Selecione disco que você deseja formatar inteiro para instalar o sistema

Aqui você seleciona como quer a sua tabela de partição, eu recomendo escolher Todos os arquivos em uma partição a menos que você saiba muito bem o que esteja fazendo

Verifique o que será mudado e se você tiver ciente das alterações no disco (que você escolheu o disco certo e tem certeza que pode ser apagado e tal) basta selecionar Finalizar o particionamento e escrever as mudanças no disco

Selecione Sim para Escrever as mudanças nos discos para continuar

Se você tivesse uma mídia adicional pro gerenciador de pacotes você declararia agora... Então basta selecionar Não para Ler mídia de instalação adicional? e Continuar

Selecione a região para os servidores mirror, no nosso caso é Brasil

E você pode selecionar qualquer um desses servidores dessa região, porém eu recomendo o debian.c3sl.ufpr.br

Aqui você pode configurar o proxy, mas deixe em branco e Continuar

Participa não...

E aqui você pode selecionar como o sistema será instalado, a minha recomendação é: ambiente de área de trabalho no Debian + utilitários de sistema padrão + LxQT.
Nada impede você de selecionar outra interface que deseje no lugar do LxQT, mas já que isso é um tutorial de como salvar pc antigo LxQT tende a ser a melhor opção

Selecione Sim para instalar o GRUB2 no disco primário

Selecione o disco novamente

E dê continuar para para reiniciar para o novo sistema, lembre-se de remover o pendrive

Se você fez tudo direitinho o sistema vai ser carregado

Basta inserir a conta de usuário padrão e senha que você vai ter acesso a área de trabalho


Pós-instalação

Adicionar seu usuário ao grupo sudo

Algumas alterações do sistema precisam de acesso administrativo, para não ter a necessidade de ficar logando como administrador sempre que for fazer uma alteração do sistema você pode adicionar o seu usuário ao grupo sudo e aí basta adicionar sudo antes do comando para executá-lo como administrador. Só precisamos acessar como administrador uma única vez para fazer essa mudança, para isso utilize o comando abaixo para acessar a conta de administrador e insira a sua senha de administrador:

su

Após inserir a sua senha de administrador use o comando abaixo para adicionar o seu usuário ao grupo sudo:

sudo usermod -aG sudo seunomedeusuário

Pronto! Agora basta acrescentar sudo antes de um comando e inserir a sua senha de usuário padrão para ter acesso administrativo.

Mudar o gerenciador de redes

Por padrão o Debian instala o ConnMan para gerenciar redes, para ter um gerenciamento mais intuitivo com ícone na barra de tarefas abra o terminal QTerminal, utilize o comando abaixo e depois tecle Enter:

sudo apt install -y network-manager nm-tray sudo systemctl disable connman.service sudo systemctl enable systemd-resolved.service NetworkManager.service

Aceleração de hardware

Por VA-API

Para aceleração de hardware nos drivers Noveau ou AMD instale o pacote mesa-va-drivers com o comando abaixo:

sudo apt install -y mesa-va-drivers

Para aceleração de hardware nos drivers Intel em computadores mais antigos instale o i965-va-driver com o comando abaixo:

sudo apt install -y i965-va-driver

Habilitando a aceleração de hardware no Firefox

Abra o Firefox, digite about:config e tecle Enter.

Mude o valor de media.ffmpeg.vaapi.enabled para true.

Mude o valor de media.ffvpx.enabled para false.

Depois utilize o comando abaixo para que o Firefox sempre inicie com aceleração de hardware:

sudo echo "MOZ_X11_EGL=1" >> /etc/environment

Melhoria das Fontes

Abra o gerenciador de arquivos PCManFM-Qt, clique na guia Exibir, ative a opção Exibir itens ocultos, acesse a pasta .config e dentro dela crie a pasta fontconfig, nessa pasta crie o arquivo fonts.conf e salve com o seguinte conteúdo:


<?xml version='1.0'?>
<!DOCTYPE fontconfig SYSTEM 'fonts.dtd'>
<fontconfig>
    <match target="font">
        <edit mode="assign" name="antialias">
            <bool>true</bool>
        </edit>
        <edit mode="assign" name="embeddedbitmap">
            <bool>false</bool>
        </edit>
        <edit mode="assign" name="hinting">
            <bool>true</bool>
        </edit>
        <edit mode="assign" name="hintstyle">
            <const>hintslight</const>
        </edit>
        <edit mode="assign" name="lcdfilter">
            <const>lcddefault</const>
        </edit>
        <edit mode="assign" name="rgba">
            <const>rgb</const>
        </edit>
    </match>
</fontconfig>

Para ver a diferença deslogue e logue novamente para surtir efeito.

Instalação do Linux Mint

Introdução

Hoje em dia a grande maioria das distribuições Linux utilizam um processo de instalação muito parecido, instalar uma distro Linux é bem rápido e simples. Aqui vou usar o Linux Mint Cinnamon de parâmetro para instalação mas não vai muito diferente disso caso você utilize outra distro voltada para usuário final.

Se tiver utilizando um notebook certifique-se de conectá-lo na tomada antes de começar

Ao inicializar vai aparecer um menu para escolha do tipo de instalação, selecione a primeira opção:

Assim que o sistema carregar selecione o ícone Install Linux Mint na área de trabalho:

Selecione a primeira opção para a instalação padrão (a menos que você entenda bastante de partições...):

Digite o seu nome, nome do computador, nome de usuário e senha (sinta-se a vontade para habilitar o login automático ou a encriptação da pasta pessoal caso deseje):

Agora é só esperar a instalação terminar...

Clique em Reiniciar Agora:

Nessa tela desconecte o pendrive do computador e aperte Enter:

Pós Instalação

A minha primeira dica é pra você deixar de ser empurrador de mouse que só busca o botão next e o botão fechar é ler, apenas ler, não ignore essa tela de boas vindas!

Altere o layout do painel caso deseje, o Mint vem com o layout moderno por padrão (estilo Windows 8/8.1/10), mas você pode deixar o tradicional (estilo Windows 7).

Após isso, você pode mudar os repositórios para acelerar o download das atualizações (recomendo os links da UFPR).

Após selecionar os links dos servidores espelho principal e base clique em OK para atualizar o cache...

...E logo em seguida instale as atualizações.

Por padrão o Mint já vem com suíte Office e navegador instalados, caso queira instalar algum outro aplicativo fique a vontade para utilizar o Gerenciador de Aplicativos do sistema pra isso:

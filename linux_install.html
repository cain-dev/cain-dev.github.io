Introdução

Hoje em dia a grande maioria das distribuições Linux utilizam um processo de instalação muito comum, hoje em dia instalar qualquer linux é bem rápido e simples. Aqui vou usar o Linux Mint Cinnamon de parâmetro para instalação mas não vai muito diferente disso caso você utilize outra distro.

Se tiver utilizando um notebook certifique-se de conectá-lo na tomada antes de começar.

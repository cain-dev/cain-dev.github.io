Instalação do Haiku

Introdução

Haiku é um sistema operacional gratuito e de código aberto compatível com o agora descontinuado BeOS. Seu desenvolvimento começou em 2001, e o sistema operacional tornou-se auto-hospedado em 2008. O primeiro lançamento alfa foi feito em setembro de 2009, e o último foi novembro de 2012; o primeiro beta foi lançado em setembro de 2018, seguido pelo beta 2 em junho de 2020 e o beta 3 em julho de 2021.

Segundo a página do Haiku:

"O Haiku é um sistema operacional de código aberto em constante desenvolvimento. Visando especificamente a computação pessoal, o Haiku é um sistema rápido, eficiente, simples de usar, fácil de aprender e, ainda assim, muito poderoso para usuários de computador de todos os níveis. Além disso, o Haiku oferece algo em relação a outras plataformas de código aberto que é bastante único: o projeto consiste em uma única equipe escrevendo tudo, desde o kernel, drivers, serviços de área de usuário, kit de ferramentas e pilha de gráficos até os aplicativos de desktop e preflets incluídos. Embora vários projetos de código aberto sejam utilizados no Haiku, eles são integrados perfeitamente. Isso permite que o Haiku alcance um nível único de consistência que fornece muitas conveniências e é realmente agradável de usar tanto por usuários finais quanto por desenvolvedores."

Em que o Haiku se destaca?

  • Foco específico na computação pessoal
  • Kernel personalizado projetado para responsividade
  • Design totalmente segmentado para grande eficiência com CPUs multiprocesso/núcleo
  • API rica orientada a objetos para desenvolvimento mais rápido
  • Sistema de arquivos database-like (BFS) com suporte para metadados indexados
  • Interface unificada e coesa

Instalando o Haiku

Embora o Haiku possa ser instalado em UEFI o procedimento é manual e pouco convidativo, portanto essa documentação não é baseada em UEFI

Essa é a tela de carregamento, apenas aguarde.

Na tela de Bem-vindo ao Haiku selecione o idioma Português (Brasil), e teclado Brasileiro (ABNT2) e em seguida clique em Instalar o Haiku:

Concorde com os termos cliando em Continuar:

Caso não haja nenhuma partição compatível com o formato Be File System (o que é muito provável) você verá um aviso te indicando a criar uma partição compatível para a instalação:

Para criar uma partição Be File System selecione Configurar partições...

Criando uma partição para o Haiku

A - Para um disco novo OU para uma nova tabela de partições:

Siga essa parte apenas se você não se importa de perder todos os dados no disco selecionado

Se o seu disco estiver sem nenhuma tabela de partição OU se você quiser criar uma nova tabela de partição e "zerar" o disco para então instalar o Haiku, selecione o disco que deseja formatar e no canto superior direito da janela selecione Inicializar > Intel Partition Map:

Você será avisado que ao fazer isso todos os dados do disco serão perdidos, clique em Continuar caso deseje instalar num disco novo ou zerar os dados do disco atual:

O sistema irá te perguntar se você quer aplicar as alterações no disco, caso tenha certeza clique em Aplicar alterações:

E caso tudo tenha dado certo, você será informado que o inicializador foi instalado com sucesso, para prosseguir clique em OK:

B - Apenas criando a partição para o Haiku sem alterar a tabela de partições atual OU após criar a nova tabela de partiçoes

Selecione o Espaço vazio no disco que você quer utilizar para instalar o Haiku, clique com o botão direito e selecione Criar...

Selecione o tamanho da partição, o Tipo de partição Be File System, marque a opção Partição Ativa e por fim clique em Criar:

Selecione Aplicar alterações:

Depois clique com o botão direito na partição que acabou de criar, selecione Formatar > Be File System:

Selecione Continuar:

E depois Formatar:

Selecione Aplicar alterações:

E caso tudo tenha dado certo dê OK:

Feche o Gestor de Discos:

E na tela do Instalador selecione a partição que acabou de criar em Para:

E depois clique em Iniciar:

Aguarde o fim da instalação...

Após o fim da instalação clique em Reiniciar:

Se tudo deu certo ao reiniciar você verá a área de trabalho do Haiku.

Pós Instalação

A minha primeira recomendação de pós instalação pra você é atualizar o sistema, para isso clique no ícone da folha no canto da tela, selecione Applications > Atualização de Software:

Espere que o programa verifique as atualizações disponíveis...

Clique em Atualizar agora para iniciar a atualização do sistema:

Aguarde a o fim da atualização...

E clique em Reiniciar:
